C++ Kinect Object Classification

Problem Breakdown

Two data dumps from an Xbox Kinect are given, providing both RGB and Depth data from each frame of the recorded video. Given this data, we aim to create a robust model capable of perofrming object recognition.

Implemented Solution

We initially detect whether or not an object is on screen by comparing the depth when no objects are shown to when an object is shown. The depth is presented as an 11-Bit value which is normalised and placed within a range of 0-255. Given no objects are on screen the depth shows a value of 0, hence making it easy to determien whether or not an object is on screen.

Using this object detection, a series of images were gathered from the video data (when an object was on screen). These images were separated out and labeled with their corresponding object which was then used to train a support vector machine.

OpenCV provides a simple to use implementation of SVM's which can be fed the images as a dataset to train the model. In doing so, the model is able to identify objects shown.

Results

Based on the types of images used (depth or RGB) the model showed different results with varying degrees of accuracy.
